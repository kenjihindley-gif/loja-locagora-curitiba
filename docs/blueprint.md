# **App Name**: Kinetic Velocity

## Core Features:

- Motorcycle Catalog Display: Showcase available motorcycles with categories (e.g., '0 KM', 'Seminovas') and promotional carousels.
- Detailed Product View: Provide comprehensive information for each motorcycle, including a image gallery, technical specifications, and pricing plans.
- Motorcycle Comparison Tool: Enable users to compare two motorcycles side-by-side, facilitating informed decisions.
- Guided Navigation: Allow seamless navigation across the application via a fixed bottom navigation bar and intuitive back/home buttons.
- WhatsApp Contact Integration: Provide a direct call-to-action button to initiate a conversation via WhatsApp for inquiries and reservations.
- Post-Reservation Checklist: Present a success screen after reservation, featuring a checklist of next steps and an optional seller code input.

## Style Guidelines:

- Primary color: Deep Ocean Blue (#214FC7), a bold and trustworthy blue for brand identity and key interactive elements. HSL: 226, 82%, 46%.
- Background color: Subtle Sky Tint (#F1F2F8), a heavily desaturated blue matching the primary's hue, for a clean and spacious feel. HSL: 226, 15%, 96%.
- Accent color: Bright Cyan Pulse (#4CCCC1F8), an analogous and vibrant cyan that stands out for calls-to-action and highlights. HSL: 196, 90%, 65%.
- Headline font: 'Space Grotesk' (sans-serif) for its modern, bold, and slightly technical aesthetic. Body and label font: 'Plus Jakarta Sans' (sans-serif) for clean readability and a contemporary feel. Note: currently only Google Fonts are supported.
- Leverage 'Material Symbols Outlined' for all icons, using fill variants for active states to ensure clarity and consistency.
- Emphasize clean, structured layouts with full-width content areas and prominent card-style elements for product displays. Utilize horizontal snap-scrolling for carousels and fixed top/bottom navigation for accessibility.
- Implement subtle transform transitions for button presses and navigation items to provide engaging feedback, alongside a gentle pulse animation for success indicators.
